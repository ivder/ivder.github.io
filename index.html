Facial Authentication

YoloV3 object detection based facial authentication. View C++ code on my github

This program will detect our face after login to Windows (Ubuntu version coming soon), then detect our face for 10 seconds. If face is detected, the program will be closed and you can continue your activity, otherwise it will foce user log-off.


Important steps :
• Dataset is annotated using labelling tool from my github repo
• YoloV3 is forked from this repository
• Training is done in Ubuntu and inference in both Windows 10 and Ubuntu

  • Pogramming Language : C++, C
  • IDE : Visual Studio 2017
  • Libraries, Framework : CUDA, OpenCV, Darknet
  • Neural Network : YoloV3-Darknet53
  • OS : Windows 10, Ubuntu
